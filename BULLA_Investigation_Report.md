## 1. ИСПОЛНИТЕЛЬНОЕ РЕЗЮМЕ

В период с 16 февраля по 12 марта 2026 года группа инсайдеров реализовала заранее спланированную схему по выводу средств из токена BULLA. Схема включала:

- Искусственный памп цены через CEX-арбитраж (февраль)
- Постепенный управляемый слив через DEX с сохранением видимости нормального рынка (февраль–март)
- Массовый скоординированный дамп пятью новыми кошельками одновременно (10–12 марта)
- Полное извлечение ликвидности LP-провайдером незадолго до финального дампа

Схема использовала архитектуру, при которой пул ликвидности с капиталом ~$500K смог обработать оборот в $8–10 млн за счёт многократного оборота средств через арбитражных ботов и розничных покупателей на CEX.

### 3.2 Фаза 1: Памп (16–24 февраля 2026)

**Механизм:**
```
1. Кошелёк 0x19113a761706... снимает BULLA с MEXC
2. Продаёт на DEX (Uniswap 0x3Ec74c0B...) → цена на DEX слегка падает
3. Арбитражный бот покупает дешёвый DEX → продаёт на CEX
4. Розница на MEXC видит рост и покупает → CEX-цена растёт
5. 0x19113a... снова снимает BULLA по более высокой цене
```

**Динамика цены:**
- 16 февраля: $0.0317 / BULLA
- 23 февраля: $0.0353 / BULLA
- 24 февраля: **$0.043 / BULLA (пик)** ← памп +35% во время продаж


#### Атака отравления адресов (Address Poisoning)

После консолидации $4.39M на коллекторе неизвестные злоумышленники (или сами операторы, создающие прикрытие) начали атаку address poisoning:

- Отправляют фейковые токены `U5DТ`, `UႽD‬Τ`, "USD Tether" (fake) на коллектор с lookalike-адресов
- Цель: запутать следственный след, заставить ошибочно отправить реальный USDT на ложный адрес
- Жертвами атак стали коллектор, агрегаторы и сами дамп-кошельки
- Реальный USDT движется только на подтверждённые адреса операторов
